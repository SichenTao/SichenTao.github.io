# Official Ranking Sources Note

This portal now uses more official ranking evidence wherever the public web allows it.

## Clarivate JCR

- `IF` and `JCR` links on publication cards point to the official Clarivate *Journal Citation Reports* journal profile page for the traced year.
- The portal resolves each journal to Clarivate's official journal abbreviation used in JCR profile URLs.
- Supporting non-Clarivate links are still preserved in generated data in case users need an alternate public reference.

## CAS Journal Ranking

- The official platform is `https://www.fenqubiao.com/`.
- Direct journal-detail views are not consistently public, so publication cards currently keep a public per-journal support page for visible quartile lookup.
- Official CAS platform materials are archived locally in this folder for traceability.

## CCF

- The portal archives official CCF materials locally, including the recommended venues PDF and saved official portal/category pages.
- These files are reference materials and do not imply that every publication card is individually labeled with a CCF class.

## Practical Note

- Official commercial or institutional portals sometimes require session handling, institutional access, or JS-driven redirects.
- For that reason, the portal keeps both:
  - the most official public-facing URL it can reliably provide, and
  - an archived local official material when available.
