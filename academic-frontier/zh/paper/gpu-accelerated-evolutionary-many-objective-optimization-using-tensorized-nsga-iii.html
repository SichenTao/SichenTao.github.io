基于张量化 NSGA-III 的 GPU 加速进化多数目标优化

作者: Hao Li, Zhenyu Liang, Ran Cheng

2025 IEEE Congress on Evolutionary Computation (CEC)

DOI 10.1109/cec65147.2025.11043108

Google ScholarOpenAlexSemantic Scholar (S2)

记录

摘要: NSGA-III 是解决多目标优化问题中应用最广泛的算法之一。然而，其基于 CPU 的设计严重限制了可扩展性和计算效率。为解决这些局限，本文提出 TensorNSGA-III，这是一个充分张量化的 NSGA-III 实现，利用 GPU 并行性进行大规模多目标优化。不同于依赖启发式近似来提升效率的传统 GPU 加速进化算法，TensorNSGA-III 在实现显著加速的同时，保持了 NSGA-III 精确的选择和变异机制。通过以张量化数据结构和优化缓存策略重构选择过程，本文方法有效消除了传统 CPU 实现和朴素 GPU 实现中的计算瓶颈。在常用数值基准上的实验结果表明，TensorNSGA-III 相比 CPU 版本 NSGA-III 最高实现 3629 倍加速。此外，研究还在多目标机器人控制任务中验证其有效性，TensorNSGA-III 能发现多样且高质量的行为解。进一步地，本文考察了大种群规模在多目标优化中的关键作用，并展示了 TensorNSGA-III 在此类场景中的可扩展性。源代码可见：https://github.com/EMI-Group/evomo。
